# 🐾 PawPerfect — Pet Adoption Platform

A secure and modern full-stack pet adoption platform built to connect pet owners and shelters with responsible adopters through a smooth and interactive experience.

## 🌐 Live Demo
🔗 https://pet-adoption-client-sigma.vercel.app

---

## 🛠️ Tech Stack

### Frontend
- Next.js (App Router)
- React.js
- Tailwind CSS
- Framer Motion

### Backend
- Node.js
- Express.js / Next.js Server Routing
- JWT Authentication
- HttpOnly Cookie Security

### Database
- MongoDB (No SQL)

### Deployment
- Vercel (Frontend+Backend)


---

## ✨ Key Features

### 🐶 Pet Management System
- Add, update, and delete pet listings
- Upload pet details with images
- Private owner dashboard

### 🔍 Smart Filtering & Search
- Advanced searching using MongoDB `$regex`
- Multi-species filtering using `$in`
- Dynamic category-based browsing

### 🔐 Secure Authentication
- JWT-based authentication
- Protected routes and private CRUD operations
- HttpOnly cookie implementation for improved security

### 🏠 Adoption Workflow
- Prevents owners from adopting their own pets
- Adoption request submission and management
- Automatic request cancellation after approval
- Real-time adoption status handling

### 🎨 Modern User Experience
- Fully responsive design
- Smooth UI animations with Framer Motion
- Light & Dark mode support
- Interactive and user-friendly interface

## 🚀 Future Improvements

- Email notifications
- Pet recommendation system
- Admin analytics dashboard
- Real-time chat between adopters and owners
